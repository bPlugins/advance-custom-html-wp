# Advanced Custom HTML Block - Feature Comparison

This document provides a single comparison table detailing the features available in the Free version versus the PRO version of the plugin.

| Feature | Free Version | PRO Version | Notes / Details |
| :--- | :---: | :---: | :--- |
| 🛡️ **EDITOR PREFERENCES** | | | |
| Editor Font Size | **Yes** | **Yes** | Adjust font sizes from 10px to 40px |
| Word Wrap | **Yes** | **Yes** | Prevents horizontal scrolling |
| Line Numbers | **Yes** | **Yes** | Shows vertical line indices |
| Syntax Languages | **Partial** | **All** | **Free:** HTML, CSS, JavaScript<br>**PRO:** PHP, JSON, Python, Java, SQL |
| Color Themes | **Partial** | **All** | **Free:** One Dark, Eclipse, Basic Light, Basic Dark<br>**PRO:** 45+ other premium themes |
| Tab Indent Size | No | **Yes** | Customize tab indent width (2, 4, 8 spaces) |
| Highlight Active Line | No | **Yes** | Highlights the row under the cursor |
| Allow Code Folding | No | **Yes** | Collapse code blocks between braces/brackets |
| Enable Code Suggestions | No | **Yes** | Automatic completion and IntelliSense |
| ⚙️ **DISPLAY & BEHAVIOR** | | | |
| Show Title Bar | **Yes** | **Yes** | Displays a header bar above the code block |
| Copy Code Button | **Yes** | **Yes** | Visitors can copy code in one click |
| Copy Button Position | No | **Yes** | **Free:** Top Right (default)<br>**PRO:** Top Right or Bottom Right |
| Copy Button Appearance | No | **Yes** | **Free:** Text button (default)<br>**PRO:** Text or SVG Icon style |
| Custom Button Texts | No | **Yes** | **Free:** "Copy" / "Copied" (default)<br>**PRO:** Custom default & success messages |
| Show Code vs. Execute | No | **Yes** | **Free:** Always executes HTML/JS<br>**PRO:** Toggle between executing code or displaying code text |
| Safe Sandbox (Iframe) | No | **Yes** | Renders scripts in an isolated sandbox to protect the site's layout |
| Sandbox Custom Sizing | No | **Yes** | Allows manual sizing of the safe sandbox area |
| 🎨 **STYLING & APPEARANCE** | | | |
| Editor Box Sizing | No | **Yes** | Custom width & height for the outer box container |
| Editor Borders & Corners | No | **Yes** | Custom border width, styling, colors, and border radius |
| Header Title Styling | No | **Yes** | Custom fonts, text size, font color, and background bar color |
| Copy Button Styling | No | **Yes** | Custom button fonts, text sizes, text colors, background colors, icon colors, and icon sizes |
| Sandbox Size Controls | No | **Yes** | Manual adjustment for sandbox width & height dimensions in styling |
